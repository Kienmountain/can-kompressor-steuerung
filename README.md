# CAN Kompressor Steuerung

Dieses Projekt steuert einen elektrischen Klimakompressor via CAN mit IXXAT USB-to-CAN FD Interface.

## Funktionen
- CAN-Kommunikation (VCI4)
- ON/OFF Steuerung
- Drehzahleingabe
- Live-Anzeige
- Logging (CSV)
